The most simple plugin to disable permanently REST API on WordPress 4.7+

== Description ==

This plugin allows to disable permanently REST API on WordPress 4.7+

== Installation ==

This section describes how to install the plugin and get it working.

1. Upload the plugin files to the `/wp-content/plugins/disable-rest-api` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the "Plugins" screen in WordPress
3. That's all folks.
